แปลงเลขฐานต่างๆ

2013-06-25

int ใช้สำหรับแปลง string แสดงเลขฐานต่างๆ ไปเป็นเลขค่าจำนวนเต็ม แล้วพิมพ์ออกมาเป็นฐานสิบได้ เช่น

hex ใช้แปลงค่าจำนวนเต็ม ไปเป็น string เลขฐานสิบหก เช่น

bin ใช้แปลงค่าจำนวนเต็มไปเป็น string เลขฐานสอง เช่น
